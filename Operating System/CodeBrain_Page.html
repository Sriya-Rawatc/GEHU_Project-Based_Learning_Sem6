About Our Project

This project reflects our vision to simplify the understanding of CPU scheduling through an interactive simulator. Using modern technologies, it compares algorithms while ensuring performance and clarity.

Designed with users in mind, the intuitive interface visualizes key metrics such as response and waiting time. Our iterative approach ensured a reliable tool for learning and better system design decisions.
